    """Decorator ‡∏™‡∏≥‡∏´‡∏£‡∏±‡∏ö‡∏à‡∏≥‡∏Å‡∏±‡∏î‡∏≠‡∏±‡∏ï‡∏£‡∏≤‡∏Å‡∏≤‡∏£‡πÄ‡∏£‡∏µ‡∏¢‡∏Å API"""
    def decorator(func):

            try:
                return func(*args, **kwargs)
            except requests.exceptions.HTTPError as e:

        return wrapper
    return decorator
